# malika-tours-in-Bayern
Simple web application about Malika Tours in Bayern ⛰️
- Responsive Design. 
- Flexbox for Columns Layouts.
- Navbar. 
- Sub Menu. 
- Animation. 
- External Icons from fontawesome.  

To run Demo : https://explorey.netlify.app/
